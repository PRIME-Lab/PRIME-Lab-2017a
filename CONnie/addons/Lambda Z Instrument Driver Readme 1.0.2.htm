3. Installation Instructions

Obtain a copy of the driver software.  It is packaged in a file with a name similar to:

                "ZPLus_LV9_102.zip"

 

A)  Use Windows Explorer and navigate to folder:

            ...\Program Files \National Instruments \LabVIEW...\instr.lib

 

B)  Create a new folder: ZPLus

 

C)  Use the ZIP utility, such as WinZip.exe, to extract the driver files

    into the "ZPLus" folder

 

D)  You will now have a "...\instr.lib\ZPLus\" folder containing:

TREE.vi

ZPL Inittialize.vi

ZPL Close.vi

Action_Status Folder

Application Folder

Configuration Folder

Data Folder

Utility Folder

six MNU menu files

"lambda ZPLus Instrument Driver readme.htm" (this file)

Prn Sc 1.jpg

 

4. Accessing the Driver in LabVIEW

There is normally no need to view the library file because the driver functions may be picked from a palette menu.

 

A)  Right-click on a blank place on the VI diagram

    See the "Functions" palette open.  Navigate:

            Functions -> 

                        Instrument I/O -> 

                                     Instrument Drivers -> 

                                                 Lambda ZPLus(+)

 

B)  Alternately, click diagram's menu bar: 

            Window -> "Show Functions Palette" and navigate the above menus


5. Using the Instrument Drivers

LabVIEW:

To verify communication with your instrument and test a typical remote

instrument operation, you should first open the Getting Started or Application Example VIs the instrument driver. Look over each of the controls and set them appropriately.  Generally, with the exception of the address field, the defaults for most controls will be sufficient for your first run. You will need to set the instrument resource name appropriately. After running the VI, check to see that reasonable data was returned and an error was not reported in the error cluster.

You can right-click any object on the driver panel to see its description.

After you have verified basic communication with your instrument, you might want to customize instrument control for your needs. If your application needs are similar to the Getting Started or Example VI, the simplest means of creating a customized VI is to save a copy of the Getting Started or Example VI by selecting "Save As" from the File menu. You can then modify this example to meet your specific needs.

For more details, reference the LabVIEW Help under:                                       Help » VI, Function, and How-To Help.
